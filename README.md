# U.S. Environmental Health Adaptation Gap Dashboard

This repository builds a GitHub Pages dashboard that maps where climate/natural hazard burden, air pollution exposure, health vulnerability, and weak adaptive capacity overlap across U.S. counties.

## Dashboard title
**U.S. Environmental Health Adaptation Gap Dashboard**

## Main research question
Which U.S. counties face the greatest gap between environmental-health risk and local adaptive capacity?

## Core contribution
Many public dashboards show environmental burden, climate hazard, or social vulnerability. This dashboard focuses on the **adaptation gap**: counties where environmental-health risks are high and the ability to cope appears weak.

## Index
The dashboard constructs the **Environmental Health Adaptation Gap Index (EHAGI)**:

```text
EHAGI = 0.30(Climate/Natural Hazard Burden)
      + 0.25(Air Pollution Exposure)
      + 0.25(Health Vulnerability)
      + 0.20(Adaptive Capacity Deficit)
```

Higher scores identify counties where environmental-health risks overlap with weaker adaptive capacity. The index is a screening tool and should not be interpreted as a causal estimate.

## Public data sources
The update pipeline uses public datasets:

1. **FEMA National Risk Index Counties** for climate/natural hazard risk, expected annual loss, social vulnerability, and community resilience.
2. **EPA AirData** for annual AQI, PM2.5, ozone, and unhealthy air days.
3. **CDC PLACES** for county-level health vulnerability measures.
4. **U.S. Census ACS 5-year API** for population and adaptive-capacity variables.
5. **Census county cartographic boundary file** for county representative points used in the map.

## How to deploy on GitHub Pages

1. Upload the contents of this folder to a GitHub repository.
2. Go to **Settings → Pages**.
3. Choose **Deploy from a branch**.
4. Set **Branch = main** and **Folder = /docs**.
5. Go to **Actions → Update dashboard data → Run workflow**.
6. After the run finishes, the dashboard will use the generated public-data file.

## Academic interpretation
The dashboard is a spatial screening and prioritization tool. It identifies counties where climate/natural hazards, pollution exposure, and health vulnerability overlap with weaker adaptive capacity. It does not prove that environmental exposure caused health outcomes.
